1 Abstract
PD model combining two complementary approaches:
•Network theory approach: Focus on the propagation of the stress aij on
the network of exposures between institutions, and the capital Ei(the
buﬀer can be used by bank ito absorb shocks).
•Credit Risk approach: Focus on the default probabilities P Diand the ten-
dency of nodes to default together (described by a Gaussian latent variable
model with correlation matrix ρ={ρij }).
- Goal: obtain a loss distribution and its quantiles.
The proposed model combine these two models: consider the system as portfolio
of risky assets, use credit risk model to calculate loss function, use network
model by considering individual bank as node in network exposure.
1
Proposed PD model: Consider matrices {aij }and {ρij }and all available
information: namely total asset Ai, capital Ei, loss given default LGDiand
probability of default P Di.
- Intuition: the default of one node increases the PD of the neighbouring nodes
in subsequent time steps according to the exposure characteristics {aij }.
- One counter-intuitive results of the model: lower correlations between nodes
correspond to higher risk.

•Credit risk approach :
The system is considered as a portfolio of investments Aiand the goal is
to obtain the estimate of risk (expressed in quantiles of the potential loss
distribution within a time period, usually 1 year).
Compared to network, the focus is P Diand nodes are intrinsically unsta-
ble and can default in absence of any external stress. To simulate default
of banks (consider tendency to default during the same time step), we use
r.v. Xi={1,...,N}to perform MC simulation for one period to get the loss
distribution and the asset kdefaults when simulated Xk=xkfalls below
a value that is a function of P Dk.
•Gaussian latent variable model (Vasicek 1987 ) :
Use a multivariate Gaussian distribution (zero mean, unit variance, cor-
relation matrix {ρij }) to sample Xi.
Condition for the default of asset k:
δk= 1 ←→ xk<Φ−1(P Dk) (2)
where Ψ is the univariate Gaussian distribution.
Then, the implied probability P Dij of double default of nodes iand j:
P Dij = Φ2(Φ−1(P Di),Φ−1(P Dj), ρij ) (3)
where Φ2is bivariate standard Gaussian distribution.
Use statistics of loss distribution to estimate the capital need to remain
solvent during the period at a conﬁdence level. Repeat the simulation for
suﬃcient times to lower the MC error below acceptable level.
Gaussian latent variable model can be seen as a ”multi-company” general-
ization of Merton model (1974). Instead of using asset return to calibrate
ρ, use equity return correlations is acceptable (daily ts data available).
